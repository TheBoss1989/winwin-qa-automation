Failed Test Details

AT-03: Filters Affect Request Failed

Steps: открыть /app, открыть фильтры, выбрать Dogs allowed, применить поиск.

Expected result: URL должен измениться и отражать применённый фильтр; последний /api/v1/offers/search request должен содержать информацию о pet/dog/filter.

Actual result: URL остался без изменений после применения фильтра.

Expected: not "https://winwin.travel/app/?search.pagination.limit=12&search.pagination.offset=0&search.stay.defaultStayPeriod.checkIn=2026-05-06T10%3A31%3A10.946Z&search.stay.defaultStayPeriod.checkOut=2026-05-09T10%3A31%3A10.947Z&search.geolocation.radius=10&search.geolocation.centerPoint.latitude=52.377956&search.geolocation.centerPoint.longitude=4.89707&search.guestQuantity.adultsQuantity=2&search.flagSearch.isSearchWithDynamicAndStaticData=true"

Impact: пользователь выбирает фильтр, но результат поиска и shareable URL могут не соответствовать выбранным условиям. Это влияет на корректность поиска и воспроизводимость результата.
